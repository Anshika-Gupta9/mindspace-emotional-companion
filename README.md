# MindSpace ✨ – Emotional Companion Web App

MindSpace is a simple mood-based journaling web application designed to help users reflect on their emotions and thoughts in a personal digital space. The app provides a clean and interactive interface where users can express their feelings and maintain a reflection journal.

## 🌟 Features
- Mood-based journaling system
- Simple and intuitive user interface
- Personal emotional reflection space
- Lightweight and responsive web design

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Purpose
The goal of MindSpace is to create a minimal and accessible platform where users can track their emotions and write journal entries in a comfortable digital environment. It explores how simple web applications can support emotional reflection and personal well-being.
